# 2.
### 문제 설명

1부터 입력받은 숫자 n 사이에 있는 소수의 개수를 반환하는 코드를 작성하시오.
소수는 1과 자기 자신으로만 나누어지는 수를 의미한다. (1은 소수가 아님)

### 제한 조건
- n은 2이상 1000000이하의 자연수

#### 출력 예
` n=10일때 4개 `   1부터 10사이의 소수는 2,3,5,7, 이니 4 반환
` n=5일때 3개`  1부터 5사이의 소수는 2,3,5 로 3반환
